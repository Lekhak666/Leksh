# 🚀 Leksh

> **Send Crypto with Style.** 💫

Leksh is a modern Web3 decentralized application (DApp) that allows users to connect their MetaMask wallet and send Ethereum transactions on the Sepolia test network.

Built using **React, Vite, Tailwind CSS, Solidity, and Ethers.js**, the platform enables seamless wallet interaction, blockchain-based transaction storage, and dynamic GIF-powered transaction cards using the Giphy API.

---

## ✨ Features

* 🔐 Connect MetaMask Wallet
* 💸 Send ETH Transactions on Sepolia Testnet
* ⛓️ Smart Contract Integration
* 🎴 Dynamic GIFs for Every Transaction
* 📜 Transaction History Display
* ⚡ Fast & Responsive UI
* 🎨 Modern Tailwind CSS Design
* 🌐 Blockchain Explorer Links
* 🔄 Real-time Wallet State Management

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone <YOUR_REPO_LINK>
```

### 2️⃣ Navigate into the project

```bash
cd leksh
```

### 3️⃣ Install dependencies

```bash
npm install
```

### 4️⃣ Create a `.env` file

```env
VITE_GIPHY_API_KEY=your_giphy_api_key
```

---

## ▶️ Run the Development Server

```bash
npm run dev
```

---

## 🔗 Smart Contract Deployment

This project uses:

* Hardhat
* Solidity Smart Contracts
* Sepolia Ethereum Testnet

To deploy the contract:

```bash
npx hardhat run scripts/deploy.js --network sepolia
```
